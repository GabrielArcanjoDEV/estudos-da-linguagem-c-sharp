## 📖 Sobre o Repositório

Bem-vindo ao meu repositório de estudos em **C# e .NET**! 🚀

Este espaço foi criado para documentar e organizar minha evolução durante a formação promovida pelo **Instituto Federal da Paraíba (IFPB)** em parceria com a **SOFT**.

Aqui estão reunidos exercícios, desafios, projetos práticos e conteúdos desenvolvidos ao longo do curso, servindo como um registro da minha evolução na programação e como uma fonte de consulta para futuros projetos.

O objetivo é aplicar, na prática, os conceitos aprendidos e construir uma base sólida para o desenvolvimento de aplicações utilizando **C# e o ecossistema .NET**.

---

## 🎯 Objetivos

Este repositório tem como principais objetivos:

* 📚 Consolidar os fundamentos da linguagem C#;
* 💻 Desenvolver a lógica de programação e resolução de problemas;
* 🧩 Aprender e aplicar os conceitos de Programação Orientada a Objetos;
* 🛠️ Desenvolver projetos e aplicações práticas;
* 📖 Documentar minha evolução durante a formação;
* 🚀 Aprimorar minhas boas práticas de desenvolvimento;
* 🔎 Criar uma base de conhecimento para futuros projetos profissionais.

---

## 📚 Conteúdos Estudados

Ao longo do repositório, serão abordados conteúdos como:

### 🔹 Fundamentos do C#

* Sintaxe e estrutura da linguagem;
* Variáveis e constantes;
* Tipos de dados;
* Operadores;
* Entrada e saída de dados;
* Conversão de tipos;
* Estruturas condicionais;
* Estruturas de repetição;
* Arrays e coleções;
* Métodos e funções.

### 🔹 Programação Orientada a Objetos (POO)

* Classes e objetos;
* Construtores;
* Encapsulamento;
* Herança;
* Polimorfismo;
* Abstração;
* Interfaces;
* Classes abstratas;
* Modificadores de acesso.

### 🔹 Desenvolvimento Prático

* Exercícios de lógica de programação;
* Desafios propostos durante a formação;
* Projetos práticos;
* Aplicação dos conceitos de C# e .NET;
* Organização e estruturação de projetos.

### 🔹 Boas Práticas

* Organização de código;
* Nomenclatura de variáveis, métodos e classes;
* Estruturação de projetos;
* Legibilidade e manutenção do código;
* Versionamento utilizando Git e GitHub.
